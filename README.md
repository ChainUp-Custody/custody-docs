# ChainUp Custody-Docs

- 文档目录说明
```
en/api-references
custody-docs
├─en                    
   ├─api-references     英文api文档
   ├─help-center        英文帮助中心
   ├─images             英文图片
   ├─introduction       英文介绍
   ├─logo               英文logo
   ├─quick-start        英文快速开始
   └─mint.json          英文配置文件
├─api-references        中文api文档
├─help-center           中文帮助中心
├─images                中文图片
├─introduction          中文介绍
├─logo                  中文logo
├─quick-start           中文快速开始
└─mint.json             中文配置文件
```

- 安装依赖
```
npm i -g mintlify
```

- 启动命令（中文） 
```
mintlify dev --port 3333
```

- 启动命令（英文）
```
cd en 

mintlify dev --port 4444
```
