---
title: 地址注册
api: POST /api/v1/address/register
description: "推送地址给平台，此地址用于分配给用户"
---

<Tip>
每次向Custody推送地址条数上限为100条，每次推送时会检查Custody剩余的地址可用数是否大于5000，如果有5000个地址没有被使用，则不允许注册；如果地址是memo类型，只需注册一个地址，memo由乙方分配
</Tip>


#### 请求参数data解密之后数据结构
<ParamField query="symbol" type="string" required>
    币种名称
</ParamField>
<ParamField query="addresses" type="string" required>
    地址列表（可以以逗号分隔，单次最多100个）
</ParamField>
<ParamField query="memo" type="bool" required>
    是否为memo类型
</ParamField>


#### 响应参数data解密之后数据结构
无
