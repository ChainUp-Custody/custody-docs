---
title: 获取指定账户信息
description: "根据币种和用户ID查询用户的账户"
---
<Note>
请注意，如果商户启用了自动归集功能，则商户下的用户账户资产将被自动转移到商户的归集账户中。
</Note>

`GET /api/v2/account/getByUidAndSymbol`

#### 请求参数数据结构
<ParamField query="time" type="long" required>
    当前时间戳
</ParamField>
<ParamField query="charset" type="string" required>
    编码格式，无特殊情况，传参数utf-8
</ParamField>
<ParamField query="version" type="string" required>
    接口版本号，无特殊情况，传参数v2
</ParamField>


<ParamField query="uid" type="string">
    用户ID
</ParamField>
<ParamField query="symbol" type="string">
    币种
</ParamField>



#### 响应参数数据结构
<Expandable title="object">
    <ResponseField name="normal_balance" type="string">正常账户余额</ResponseField>
    <ResponseField name="lock_balance" type="string">冻结账户余额</ResponseField>
    <ResponseField name="deposit_address" type="string">币种对应的充值地址</ResponseField>
</Expandable>
