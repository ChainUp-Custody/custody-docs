## III. 创建并配置企业钱包

- 登录ChainUp Custody平台

浏览器打开[ChainUp Custody平台](https://custody.ChainUp.com/)， 使用【Custody】应用，首页或云端钱包【扫码】功能扫描二维码登录

- 创建钱包

进入【ChainUp Custody】页面，创建钱包（钱包名称不支持修改，请谨慎填写）

- 开启币种

1. 开启币种前，请先联系平台开通区块链网络的原生代币（开通区块链网络的原生代币后，对应代币即可免费使用，若未收录可联系平台免费上币)

2. 开通币种后，进入钱包点击【币种管理】，搜索币种

3. 进入编辑页面，开启币种


### 获取API信息

1. 创建API（目前仅支持一个全功能权限的API）

2. 创建完成，完善公钥信息

3. 开发测试，根据平台提供的技术方案与API文档进行系统对接
