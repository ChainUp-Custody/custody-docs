---
title: 归集币种设置
api: POST /api/mpc/billing/auto_collect/symbol/set
description: "设置币种的最小归集金额、以及加油的最大矿工费"
---

#### 请求参数数据结构
<ParamField query="symbol" type="string" required>
    币种唯一标识，例：`USDTERC20`
</ParamField>
<ParamField query="collect_min" type="string" required>
    归集最小币种金额；最大6位小数精度，不超过9999999999999999
</ParamField>
<ParamField query="fueling_limit" type="string" required>
    归集最多消耗矿工费金额；最大6位小数精度，不超过9999999999999999
</ParamField>


#### 响应参数数据结构
<Expandable title="object">
    <ResponseField name="symbol" type="string">配置完成币种唯一标识</ResponseField>
</Expandable>
