---
title: 获取NFT信息
api: POST /api/nft/info
description: "获取NFT图片，名称等数据"
---

#### 请求参数数据结构
<ParamField query="base_symbol" type="long" required>
    区块链网络的原生代币种名称，币种唯一标识，例：`ETH`
</ParamField>
<ParamField query="contract_address" type="string" required>
    NFT币种合约地址
</ParamField>
<ParamField query="token_id" type="string">
    NFT token id
</ParamField>



#### 响应参数数据结构
<ResponseField name="base_symbol" type="string">区块链网络的原生代币种名称，币种唯一标识，例：`ETH`</ResponseField>
<ResponseField name="contract_address" type="string">NFT币种合约地址</ResponseField>
<ResponseField name="token_id" type="string">NFT token id</ResponseField>
<ResponseField name="nft_name" type="string">NFT 名称, 不存在时, 使用 "# + tokenId", 如: #1000001</ResponseField>
<ResponseField name="image" type="string">nft图片或视频url, 或图片base64</ResponseField>
<ResponseField name="image_type" type="string">nft图片或视频类型, 如：".png", ".mp4"</ResponseField>
<ResponseField name="is_video" type="bool">是否是视频, `true`: image是视频, 反之不是视频</ResponseField>
