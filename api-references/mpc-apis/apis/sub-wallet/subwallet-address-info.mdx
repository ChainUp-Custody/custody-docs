---
title: 验证地址信息
api: GET /api/mpc/sub_wallet/address/info
description: "查询地址对应在系统中的用户信息"
---

#### 请求参数数据结构
<ParamField query="address" type="string" required>
    任意地址
</ParamField>
<ParamField query="memo" type="string">
    Memo类型需要传入memo
</ParamField>



#### 响应参数数据结构
<Expandable title="object">
    <ResponseField name="sub_wallet_id" type="integer">钱包id</ResponseField>
    <ResponseField name="addr_type" type="integer">地址类型，1：用户地址，2：系统地址(包括归集地址、找零地址)。`系统地址`不可给分配给用户使用, UTXO交易的找零都会找零到找零地址</ResponseField>
    <ResponseField name="merge_address_symbol" type="string">币种唯一标识</ResponseField>
</Expandable>
