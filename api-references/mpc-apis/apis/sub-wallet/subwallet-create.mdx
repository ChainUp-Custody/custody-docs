---
title: 创建MPC钱包
api: POST /api/mpc/sub_wallet/create
description: "为工作站创建一个新钱包"
---

#### 请求参数数据结构
<ParamField query="sub_wallet_name" type="string" required>
    钱包名称，`最大支持50个字符`，同一钱包下钱包不能重名，例：mpc矿池
</ParamField>
<ParamField query="app_show_status" type="string">
    创建后app和web端是否展示该钱包，`1`展示, `2`不展示；默认不展示，展示后可以在app和web端进行充提币、web3交易等操作
</ParamField>



#### 响应参数数据结构
<Expandable title="object">
    <ResponseField name="sub_wallet_id" type="integer">钱包id</ResponseField>
</Expandable>
