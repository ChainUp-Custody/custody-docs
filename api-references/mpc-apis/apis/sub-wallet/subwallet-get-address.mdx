---
title: 查询地址
api: POST /api/mpc/sub_wallet/get/address/list
description: "查询钱包下对应币种的地址信息"
---

#### 请求参数数据结构
<ParamField query="sub_wallet_id" type="integer" required>
    钱包id
</ParamField>
<ParamField query="symbol" type="string" required>
    币种唯一标识，例：`USDTERC20`
</ParamField>
<ParamField query="max_id" type="integer" required>
    地址起始id, Memo类型币种起始id为memo，默认0
</ParamField>


#### 响应参数数据结构
<ResponseField type="object[ ]">
    <Expandable title="object">
        <ResponseField name="address" type="string">创建的地址</ResponseField>
        <ResponseField name="addr_type" type="integer">地址类型，1：用户地址，2：系统地址(包括归集地址、找零地址)。`系统地址`不可给分配给用户使用, UTXO交易的找零都会找零到找零地址</ResponseField>
        <ResponseField name="memo" type="string"></ResponseField>
    </Expandable>
</ResponseField>
