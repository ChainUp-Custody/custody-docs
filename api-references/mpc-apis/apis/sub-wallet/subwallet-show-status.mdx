---
title: 修改钱包状态
api: POST /api/mpc/sub_wallet/change_show_status
description: "指定钱包在app、web客户端的展示情况，不展示无法在app发起交易"
---

#### 请求参数数据结构
<ParamField query="sub_wallet_ids" type="string" required>
    多个钱包id的字符串，英文逗号分割
</ParamField>
<ParamField query="app_show_status" type="string" required>
    创建后app和web端是否展示该钱包，`1`展示, `2`不展示；默认不展示
</ParamField>



#### 响应参数数据结构
无
