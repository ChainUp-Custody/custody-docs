## I. TRON资源作用
Tron 区块链采用独特的费用模型，与以太坊等区块链不同。它不使用固定的 Gas 费，而是通过带宽和能量两种资源完成交易。
若钱包没有交易所需资源，用户需冻结TRX获取资源，通过直接燃烧地址的 TRX获取资源，以支持波场网络的交易活动。


  
## II. 平台介绍
质押和直接直接燃烧TRX支付矿工费的成本较高。为了帮助客户节省交易费用并降低运营成本，Custody平台提供了资源交易功能，客户可以通过API直接购买TRON资源，实现能量秒到账，直接节省手续费。
  
**高效：** API租赁资源，流程快捷方便，不限制笔数，不限制时间，实时获取能量，Tron 波场用户必备省钱专家。

**专业：** Custody提供24小时专属客服服务，以及专业技术团队支持，来为所有类型的外部客户提供值得信赖的租赁服务。

**经济：** Custody宗旨是为每位波场交易者尽力降低波场交易费用，购买量越大，优惠力度越大。

  
**一站式操作：** 只要您提供的地址拥有足够的资源，Custody将自动将这些资源覆盖到工作站上的每一笔波场交易。无需再消耗任何TRX，整个交易过程简便高效，操作复杂性大幅降低，助您专注于业务发展，轻松实现高效交易。
  
  <Tip> 
  只有MPC托管客户才可以采用这种方案。
  </Tip>

  ### 二.客户授权平台
