## I. TRON资源作用
Tron 区块链采用独特的费用模型，与以太坊等区块链不同。它不使用固定的 Gas 费，而是通过带宽和能量两种资源完成交易。
若钱包没有交易所需资源，用户需冻结TRX获取资源，通过直接燃烧地址的 TRX获取资源，以支持波场网络的交易活动。


  
## II. 平台介绍
质押和直接直接燃烧TRX支付矿工费的成本较高。为了帮助客户节省交易费用并降低运营成本，Custody平台提供了资源交易功能，客户可以通过API直接购买TRON资源，实现能量秒到账，直接节省手续费。
  
**高效：** API租赁资源，流程快捷方便，不限制笔数，不限制时间，实时获取能量，Tron 波场用户必备省钱专家。

**专业：** Custody提供24小时专属客服服务，以及专业技术团队支持，来为所有类型的外部客户提供值得信赖的租赁服务。

**经济：** Custody宗旨是为每位波场交易者尽力降低波场交易费用，购买量越大，优惠力度越大。


  
## III. 接入流程

### 一.创建MPC工作站

请参考以下链接下载 Custody App，并使用手机号或邮箱注册 Custody 账号。完成注册后，您可以访问 MPC 系统，创建一个工作站。

[下载 Custody App](https://custody.chainup.com/login)

[创建MPC工作站](/user-guide/mpc-wallet/wallet/create-wallet)

<Note>
对于非托管客户，您的 MPC 工作站采用特殊的续费模式。请联系 Custody 客服团队，并提供您的 MPC 工作站名称（ID），同时告知您是因购买 TRON 资源而创建的工作站。客服团队将为您的工作站调整配置和使用期限。
</Note>


  
  ### 二.客户授权平台
  
请向 MPC 工作站下的 Web3 钱包**充值至少 101 个 TRX**，发起自动代理授权申请。

<Note>
在不修改策略引擎的前提下，工作站创建者可以直接发起申请，并使用私钥签名完成授权。
</Note>
  
  途径一：登录 MPC 工作站，进入 [更多] - [工具箱]，访问 TRON 自动代理工具。

  ### 三.联系Custody完成费用配置
  
  
授权完成后，访问 MPC工作站 - 工具 - TRON自动代理，如果工具状态显示**等待系统配置**，说明授权成功，请联系Custody团队完成资源价格费率配置。

<Tip> 
由于资源的价格会根据您的使用量和使用时长进行动态调整，具有高度个性化和灵活性，因此没有统一的收费标准。Custody系统将根据您的使用需求，与您沟通并确定资源优惠费率。
</Tip>
  
  ### 四.完成API配置

目前Custody只支持通过API购买资源，请访问  MPC工作站 - 开发者 - API管理，完成API的配置。

具体流程可以参考[API配置](/user-guide/mpc-wallet/api-integration/api)。

<Note>
请注意，对于仅调用 API 而不在工作站进行交易的特殊客户（如使用 API 购买 TRON 资源的客户），无需部署 API Co-Signer。
</Note>

  ### 五.调用Custody API购买TRON资源

  购买资源请参考API文档：[购买TRON网络交易资源](/api-references/mpc-apis/apis/tron/delegate-create)

  获取购买记录请参考API文档：[获取资源购买记录](/api-references/mpc-apis/apis/tron/delegate-record-list)

  同步购买记录请参考API文档：[同步资源购买记录](/api-references/mpc-apis/apis/tron/delegate-record-sync-list)

  购买记录交易通知请参考API文档：[TRON购买资源交易通知](/api-references/mpc-apis/notify/tron)



## VI.常见问题
  
  <Accordion title="如何了解订单实际消耗的交易资源">
  请访问区块链浏览器，通过查询交易的TXID，了解交易消耗的资源数量。
  </Accordion>
  
<Accordion title="系统如何计费和结算代理服务费？">
Custody系统会根据实际消耗的资源和实时价格，每日0点（UTC+8）统计前一日的消费数据，并在每日17:00（UTC+8）从您的授权钱包结算相关费用。
</Accordion>
