---
title: 创建Web3交易
api: POST /api/mpc/web3/trans/create
description: "通过API创建一笔Web3交易"
---

目前Keysecure支持的Web3交易网络如下：

- Ethereum

- BNB Smart Chain

- HECO

- Avalanche (C-Chain)

- Polygon

- Filecoin

- TRON

- Wizarre Scroll


#### 请求参数数据结构
<ParamField query="sub_wallet_id" type="integer" required>
    钱包id
</ParamField>
<ParamField query="request_id" type="string" required>
   	转账唯一标识
</ParamField>
<ParamField query="main_chain_symbol" type="string" required>
    网络原生代币币名，币种唯一标识，例：`ETH`
</ParamField>
<ParamField query="from" type="string">
    交易发起地址，钱包存在多地址时, 可指定交易地址, 不指定时默认使用钱包“常用地址“
</ParamField>
<ParamField query="interactive_contract" type="string" required>
    交互合约
</ParamField>
<ParamField query="amount" type="string" required>
   	交易金额，-1:表示无穷大, 该金额表示网络原生代币使用金额, 如金额为1ETH, 传值为:1
</ParamField>
<ParamField query="gas_price" type="string" required>
    gas 矿工费，单位：Gwei
</ParamField>
<ParamField query="gas_limit" type="string" required>
   	gas limit 矿工费
</ParamField>
<ParamField query="input_data" type="string" required>
   	合约交易的方法参数组成的16进制数据
</ParamField>
<ParamField query="trans_type" type="string" required>
   	0:授权交易，1：其他交易。为0时，amount设置金额无效
</ParamField>
<ParamField query="dapp_name" type="string">
   	dapp名称
</ParamField>
<ParamField query="dapp_url" type="string">
   	dapp 访问url
</ParamField>
<ParamField query="dapp_img" type="string">
   	dapp 图片
</ParamField>
<ParamField query="sign" type="string">
   	使用RSA私钥签名, 参与签名的参数: "request_id", "sub_wallet_id", "main_chain_symbol", "interactive_contract", "amount", "input_data"。签名规则详见[Co-Signer交易签名验证](/api-references/mpc-apis/co-signer/sign-verify)
</ParamField>



#### 响应参数数据结构
<Expandable title="object">
    <ResponseField name="trans_id" type="integer">Web3交易ID</ResponseField>
</Expandable>
