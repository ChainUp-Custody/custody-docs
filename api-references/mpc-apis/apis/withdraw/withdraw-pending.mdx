description: "转账签名完成后，由于矿工费不足而长时间未上链的时候，可以重新指定更高额的矿工费用加速上链"

#### 请求参数数据结构
<ParamField query="withdraw_id" type="integer" required>
    转账id
</ParamField>
<ParamField query="fee_rate" type="string">
   	UTXO网络类型费率；注意：不要和其他系列参数一起传入
</ParamField>
<ParamField query="size" type="string">
   	UTXO网络类型字节大小；注意：不要和其他系列参数一起传入
</ParamField>
<ParamField query="gas_price" type="string">
     EVM网络类型gas推荐单价；注意：不要和其他系列参数一起传入
</ParamField>
<ParamField query="gas_limit" type="string">
   	EVM网络类型gas限制；注意：不要和其他系列参数一起传入
</ParamField>
<ParamField query="fee" type="string">
    其他系列，如DOT\TRX，使用fee；注意：不要和其他系列参数一起传入；
</ParamField>
