API Co-Signer可以帮助用户实现审批和签名自动化，非常适合交易量大或活动频繁的工作站。如果您希望实现自动化交易，它是一个理想的选择。

* Co-Signer在交易签名前会回调您的业务系统，得到授权通过后才能发起签名，否则无法进行交易。

* 归集交易在app端签名授权后，将不会回调客户系统

* 查看[Co-Signer部署文档](/api-references/mpc-apis/co-signer/deploy)
