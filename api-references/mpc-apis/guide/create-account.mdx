客户可自助完成Keysecure  MPC钱包的开户操作。



## I. 下载APP

用户可通过以下方式获取APP：

手机浏览器访问[官方网站](https://www.keysecure.io/)或使用手机扫描二维码

## II. 注册登录

系统支持邮箱、手机号两种方式注册登录；注册与登录功能合二为一，首次登录会自动完成注册；



## III. 创建工作站

登录系统后，为团队在APP创建一个`MPC工作站`，工作站创建成功将会在APP本地生成其中一份私钥分片，客户务必及时备份私钥分片，未备份私钥分片部分功能将不可用。

目前系统支持`手动抄写`、`上传云服务器`两种备份方式。

MPC的系统分为`工作站`、`钱包`两个类型；

<Tip>
一个工作站可以创建多个钱包
</Tip>


### 应用场景

* 工作站可用于某个业务线的资产统计；钱包分配给**不同的用户**使用。

* 工作站可用于公司总资产的统计；钱包分配给**不同的业务部门**使用。

<Warning>
**私钥密码**用于**加解密私钥分片**，**非常重要**，为了您的资产安全**请务必及时备份**。 未备份私钥分片和密码将限制您MPC 钱包功能的使用。
</Warning>

* Keysecure 采用的是**3-3的签名策略**，其中2个私钥分片分别加密存储在微软云服务商和亚马逊云服务商，另外1个私钥分片在**用户本地**；

* **用户对资产有100%的控制权**：Keysecure **仅协助**用户管理资产，在没有用户私钥分片签名情况下，无法触碰用户资产。因此用户需要及时将**本地私钥分片进行备份并妥善保管**；

* 为了便于用户记忆，私钥分片以**助记词**形式存在。

若工作站配置不满足您的需求，可联系商务经理。
