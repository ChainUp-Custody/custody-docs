`POST /客户回调通知地址`

#### I. 交易通知请求参数统一格式

| Param  | 类型   | 是否必须 | 说明                                                    |
| :----- | :----- | :------- | :------------------------------------------------------ |
| app_id | String | 是     | 商户唯一标识                                            |
| data   | String | 是     | 加密之后的字符串，具体加密信息见各接口 `请求参数数据结构` |

#### II. 交易通知响应参数统一格式

返回字符串 （注意此处返回参数**无需进行加密**）：
* `SUCCESS`表示成功
* `FAILURE`表示失败
