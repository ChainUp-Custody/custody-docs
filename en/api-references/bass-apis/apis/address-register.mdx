---
title: Register Address
api: POST /api/v1/address/register
description: "Push address to the platform, this address is used to assign to users"
---

<Tip>
A maximum of 100 addresses per push to Custody, with each push checking that the number of addresses remaining available to Custody is greater than 5,000 and disallowing registration if 5,000 addresses are not being used. If the address is memo type, only one address needs to be registered, and the memo will be allocated by Party B
</Tip>


#### Decrypted Request Parameters
<ParamField query="symbol" type="string" required>
    Token name
</ParamField>
<ParamField query="addresses" type="string" required>
    Address list (can be separated by commas, up to 100 at a time)
</ParamField>
<ParamField query="memo" type="bool" required>
    Whether it is memo type
</ParamField>


#### Decrypted Response Data Parameters
None
