---
title: Account Information for Specified User
description: "Search the user’s account according to the coin and user ID. "
---
<Note>
If the merchant starts auto sweeping operation, the user account will be transferred to the merchant’s omnibus wallet
</Note>

`GET /api/v2/account/getByUidAndSymbol`

#### Decrypted Request Parameters
<ParamField query="time" type="long" required>
    current timestamp
</ParamField>
<ParamField query="charset" type="string" required>
    encoding format, no special case, pass parameter UTF-8
</ParamField>
<ParamField query="version" type="string" required>
    interface version number, no special case, pass parameter v2
</ParamField>


<ParamField query="uid" type="string">
    user ID
</ParamField>
<ParamField query="symbol" type="string">
    coin
</ParamField>



#### Response Data Parameters
<Expandable title="object">
    <ResponseField name="normal_balance" type="string">normal account balance</ResponseField>
    <ResponseField name="lock_balance" type="string">freeze account balance</ResponseField>
    <ResponseField name="deposit_address" type="string">deposit_address</ResponseField>
</Expandable>
