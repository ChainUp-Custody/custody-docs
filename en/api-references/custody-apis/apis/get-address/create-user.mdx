---
title: User Mobile Registration
description: "Register to become a wallet user"
---

`POST /api/v2/user/createUser`

<Note>
Please note that the default registration limit is 50,000 users. If the number of users exceeds this limit, please contact your dedicated business manager or custody support for assistance.
</Note>
