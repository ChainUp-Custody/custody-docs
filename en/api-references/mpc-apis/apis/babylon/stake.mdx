#### Request Parameters
<ParamField query="sub_wallet_id" type="integer" required>
    Wallet ID
</ParamField>
<ParamField query="lock_number" type="integer" required>
    Staking period (block height)

    If you need to test the Unbond, you need to fill in the lock_number and amount according to the official requirements of **babylon**; if you need to test the Unstake, you can set the lock_number smaller, and [the amount > the current mining fee + the minimum withdrawal amount]
</ParamField>
<ParamField query="symbol" type="string" required>
    Unique identifier for the coin, e.g.: `BTC`
</ParamField>
<ParamField query="amount" type="string" required>
   	Stake amount
</ParamField>
<ParamField query="fp_key" type="string" required>
   	Finality Provider
</ParamField>
<ParamField query="request_id" type="string" required>
   	The unique identifier for stake
</ParamField>
<ParamField query="fee" type="string">
   	Customized fee, e.g.: `0.021`
</ParamField>
