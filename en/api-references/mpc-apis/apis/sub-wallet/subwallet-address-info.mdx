---
title: Verify Address Information
api: GET /api/mpc/sub_wallet/address/info
description: "Input a specific address and get the response of the corresponding custody user and coin information"
---

#### Request Parameter Structure
<ParamField query="address" type="string" required>
    Any address
</ParamField>
<ParamField query="memo" type="string">
    If it's a Memo type, input the memo
</ParamField>


#### Response Data Parameters
<Expandable title="object">
    <ResponseField name="sub_wallet_id" type="integer">Wallet ID</ResponseField>
    <ResponseField name="addr_type" type="integer">Address type, 1: User address, 2: System address (including collection address, change address). `System address` cannot be assigned to users, change from UTXO transactions will all go to the change address</ResponseField>
    <ResponseField name="merge_address_symbol" type="string">Unique coin identifier</ResponseField>
</Expandable>
