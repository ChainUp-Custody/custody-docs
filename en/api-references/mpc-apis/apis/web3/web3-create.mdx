---
title: Create Web3 Transaction
api: POST /api/mpc/web3/trans/create
description: "Create a Web3 transaction."
---

#### Request Parameters
<ParamField query="sub_wallet_id" type="integer" required>
    Sub wallet id
</ParamField>
<ParamField query="request_id" type="string" required>
   	The unique identifier for transferring coins
</ParamField>
<ParamField query="main_chain_symbol" type="string" required>
    Main chain coin symbol, Unique identifier for the coin, e.g.：`USDTERC20`
</ParamField>
<ParamField query="from" type="string">
    Transaction initiation address. In the case of multiple addresses in the sub-wallet, you can specify the transaction address. If not specified, the sub-wallet's "commonly used address" will be used by default.
</ParamField>
<ParamField query="interactive_contract" type="string" required>
    Interactive contract
</ParamField>
<ParamField query="amount" type="string" required>
   	Transaction amount (-1 for infinite)
</ParamField>
<ParamField query="gas_price" type="string" required>
    Gas fee，unit：Gwei
</ParamField>
<ParamField query="gas_limit" type="string" required>
   	Gas limit fee
</ParamField>
<ParamField query="input_data" type="string" required>
   	Hexadecimal data for contract transaction
</ParamField>
<ParamField query="trans_type" type="string" required>
   	0: Authorization transaction, 1: Other transaction. If 0, the amount field is invalid.
</ParamField>
<ParamField query="dapp_name" type="string">
   	Dapp name
</ParamField>
<ParamField query="dapp_url" type="string">
   	Dapp URL
</ParamField>
<ParamField query="dapp_img" type="string">
   	Dapp image
</ParamField>
<ParamField query="sign" type="string">
   	RSA private key signature. Parameters involved in the signature: "request_id", "sub_wallet_id", "main_chain_symbol", "interactive_contract", "amount", "input_data". Refer to [`Co-Signer Transaction Signature Verification`](/api-references/mpc-apis/co-signer/sign-verify) for detailed signing rules.
</ParamField>



#### Response Data Parameters
<Expandable title="object">
    <ResponseField name="trans_id" type="integer">Web3 transaction ID</ResponseField>
</Expandable>
