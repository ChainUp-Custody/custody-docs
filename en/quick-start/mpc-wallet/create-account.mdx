##  Create an Account

If you are using CryptSafe's services for the first time, you need to register an account. Your account can be used across all services.

CryptSafe supports registration via mobile number or email.

##  2FA (Two-Factor Authentication)

    <img src="/images/quickstart/2fa.png" width="200"/>
</div>

For the security of your assets, we require you to bind the Google Authenticator. It is used for verification in certain operations.

<Warning>
If you do not bind it, some features will be restricted.
</Warning>
