## I. Function Introduction

Whitelist management is a function to enhance the security of withdrawal transactions. By restricting withdrawals to only be sent to pre-set whitelist addresses, it effectively controls the flow of funds and ensures the safety of funds.

## II. Whitelist Function Operation Procedure

To enhance the security of withdrawal transactions, you need to perform the following operations:

<Step title="Enable Whitelist Function">
The wallet creator needs to turn on the whitelist switch for the corresponding wallet in the wallet settings.
<Note>
After enabling the function, the withdrawal operation will only support going to the whitelist address, except for API withdrawals.
</Note>

## III. Frequently Asked Questions

<Accordion title="Why do I need to enable the whitelist function?">
Enabling the whitelist function can limit withdrawal operations to only be sent to pre-set whitelist addresses, effectively controlling the flow of funds and enhancing the security of funds.
</Accordion>

<Accordion title="How do I add or delete whitelist addresses?">
You can add or delete whitelist addresses on the whitelist management page, ensuring that only trusted addresses are included in the whitelist.
</Accordion>
