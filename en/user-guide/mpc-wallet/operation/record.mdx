## I. Function Introduction

In the ChainUp Custody wallet, you can view, filter, and export transaction information you have access to. You can filter and search for transaction information of the main wallet based on coin name, NFT collection name, contract address, sub-wallet, and Coin network.

## II. Operation Procedure

<Steps>
<Step title="View Transaction Information">
In the **APP**, you can click on [Historical Bills] on the [Assets] page of the specified sub-wallet to view and filter, or view the transaction information of the coin in [Coin Details].

<Step title="Export Transaction Information">
On the **Web Page**, you can download the required transaction information by clicking the export button.

**The download information fields include:**

- TXID/Transaction Information

- Transfer ID/Order Number

- NFT Token Name/Name

- Main Chain/Main Chain Name

- Amount/Amount

- From/Transaction Initiation Address

## III. Frequently Asked Questions

<Accordion title="How do I view the transaction information of other main wallets?">
You can click the MPC button at the top of the page, or click the main wallet switch button on the left, switch to another main wallet, and then enter the transaction information module of this main wallet for viewing.
</Accordion>

<Accordion title="Why can't I view or search for a certain transaction information?">
Please confirm the following points:

1. Do you have permission for the transaction: You need to have the co-management authority of the relevant main wallet/sub-wallet;
2. If it is a transfer transaction, please confirm whether this transaction has created an order number;
3. If it is a receiving transaction, please confirm the TXID of the order and whether the Custody system supports the currency;
4. If you want to view Fil related information: Sorry, we temporarily do not support users to view Fil related information on the MPC wallet Web platform.

If there are still related problems, please contact online customer service for help.
</Accordion>

<Accordion title="Why does some transaction information seem incomplete?">
Possible reasons include:

1. The system cannot fully parse the transaction information field of the contract call class;
2. Some fields are associated with transaction status: for example, only transactions that have been successfully chained will display miner fees.

You can contact online customer service for more help.
</Accordion>
