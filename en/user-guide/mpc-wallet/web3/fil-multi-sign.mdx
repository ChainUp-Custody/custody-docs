##  Function Introduction

The Filcoin mainnet provides a function that supports multiple accounts to manage a private key-less wallet, this function is called the FIL multisig wallet. Users can effectively manage this wallet by initiating proposals and voting on proposals.

**FIL multisig wallet is suitable for the following business scenarios:**

- **Multiple shareholders jointly manage FIL assets:**

Through the multisig wallet, multiple different shareholders can jointly manage FIL assets. Manage and control your FIL assets scientifically based on the set number of shareholders and threshold.

- **Multiple members manage FIL nodes:**

You can use the multisig wallet as the owner or beneficiary of the FIL node, realizing the function of multiple members jointly managing the FIL node.

<Note>
If you want to learn more detailed terms about FIL and the transaction process of the FIL multisig wallet, please refer to [FIL Multisig Wallet Management](help-center/manage-mpc/web3/fil-multi-sign)
</Note>

### Manage Multisig Wallet Signers and Threshold

Keysecure's MPC Workspace allows users to publish and edit proposals for the signers and threshold of the multisig wallet, and can vote on these proposals.

<Steps>
<Step title="Edit signers">

You can choose to add or reduce signers; after selecting and entering the corresponding parameters, the transaction will be on-chain, and the proposal will be successfully initiated.

##  Frequently Asked Questions

<Accordion title="Can I set multiple signers in one wallet?">
Yes, each multisig wallet can have multiple signers. The number of signers and the threshold can be set when creating the multisig wallet, or they can be modified later through proposals.
</Accordion>

<Accordion title="If a signer is removed, can they be added again?">
Yes, if a signer is removed, they can be added again. Please note that only the owner of the multisig wallet can add or remove signers.
</Accordion>

<Accordion title="Can I change the threshold of the multisig wallet?">
Yes, you can change the threshold of the multisig wallet. Changing the threshold needs to be done through a proposal, and it needs the consent of the signers that reach the current threshold to change.
</Accordion>

<Accordion title="If I am not the owner of the multisig wallet, can I initiate a proposal?">
Yes, all signers of the multisig wallet can initiate proposals. However, the proposal needs to reach the set threshold to be executed.
</Accordion>
