---

## 概述

CryptSafe 全托管系统提供了一套灵活且安全的账户架构，旨在满足不同类型用户和业务线的需求，包括个人用户和企业钱包。该架构允许用户创建并管理多种类型的钱包，每种钱包支持多个币种，从而提供了高度的灵活性和可扩展性。

## 业务逻辑

在 CryptSafe 的账户架构中，一个用户账户可以创建一个个人钱包以及多个企业钱包。每个钱包都能够支持多种数字资产，从而满足不同场景下的资产管理需求。

### 个人钱包：

- **功能**：个人钱包专为管理个人资产而设计，不支持 API 功能。这意味着个人用户无法通过 API 进行自动化的交易操作。


- **特点**：
  - **联盟转账**：个人钱包支持与企业钱包之间的联盟转账，且这类转账不收取手续费。这使得用户能够在个人和企业资产之间进行便捷的资金调动。
  - **安全管理**：个人钱包提供了高度的安全性，确保个人资产得到妥善保护。

### 企业钱包：

- **功能**：企业钱包不仅支持团队管理和多种企业级功能，还可以通过 API 进行交易和管理。这使得企业能够实现自动化的操作，提高效率和管理水平。


- **特点**：
  - **团队管理**：企业钱包允许多个团队成员共同管理钱包，提供了权限分配和角色管理的功能，从而确保企业资产的安全和管理的有序。
  - **KYT（Know Your Transaction）**：企业钱包集成了 KYT 功能，用于监控和分析交易，帮助企业遵守反洗钱法规并降低风险。
  - **API 支持**：通过 API，企业钱包能够自动化地分配币种地址给终端用户，进行交易、查询余额等操作。这为企业提供了高效的资产管理解决方案。
