<Accordion title="行业领先安全架构">
ChainUp Custody采用下一代技术构建，致力于解决未来运营和安全挑战。我们采用多层安全矩阵，包括MPC、Intel SGX技术，以及我们独有的签名策略引擎和存款地址身份验证网络，为客户提供市场上最坚固、最难攻破的系统保障。

<Accordion title="简易集成的SDK">
ChainUp Custody托管钱包搭载了便捷易用的SDK，助易于使用的 API 和即插即用的 SDK，开发人员可以迅速熟悉ChainUp Custody服务并将其顺利融入到他们的应用程序中，减少钱包开发所需的工作量。

<Accordion title="灵活的部署模型">
ChainUp Custody提供多种部署选项，包括SaaS、私有云和客户托管，能够灵活满足不同需求。我们利用MPC（多方计算）或HSM（硬件安全模块）来进行密钥管理，以灵活地部署在云中或客户托管的环境中，以满足您的风险、监管和合规性需求，而无需牺牲安全性或可扩展性。
</Accordion>
<Accordion title="多区块链支持">
ChainUp Custody提供多区块链支持，涵盖超过200个主链和1000多种代币。我们还支持 Web3 和领先的 NFT 代币标准，比如 ERC、BEP、SPL 等。这些功能使我们能够满足多样化的区块链需求，并为用户提供广泛的选择。
</Accordion>
<Accordion title="强大的策略引擎">
为适应您的工作流程，ChainUp Custody提供自定义用户权限功能，有效地分隔用户权限，同时允许用户配置影响交易处理和批准方式的风控策略，以确保资产安全，防止内部作恶和误操作导致的资金安全风险。
</Accordion>
<Accordion title="KYT与AML集成">
ChainUp Custody与行业领先的KYT数字资产合规解决方案合作，直接从ChainUp Custody平台提供实时交易监控。每笔交易都会根据您配置的合规策略规则和合规提供商的数据进行自动筛选，以避免您的团队将交易发送至有风险的钱包。合规团队还可以针对潜在风险钱包的充值交易进行冻结，以在使用资金之前进行合规性评估。
</Accordion>
