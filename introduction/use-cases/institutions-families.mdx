---
title: 机构和家族办公室
---

ChainUp Custody帮助机构和家族办公室安全的管理大规模数字资产，提供合规性解决方案，降低风险并简化资产管理流程，获得透明的交易记录和审计跟踪，以满足监管要求。ChainUp Custody的解决方案低成本、易于使用，有助于机构和家族办公室在数字资产领域取得成功。

<Accordion title="企业级安全架构">
ChainUp Custody运用最新的MPC加密技术与硬件隔离相结合的多层技术，保护客户和投资者的资金免受网络攻击、内部共谋和人为错误的影响。
</Accordion>
<Accordion title="简易集成的SDK">
ChainUp Custody托管钱包搭载了便捷易用的SDK，助易于使用的 API 和即插即用的 SDK，开发人员可以迅速熟悉ChainUp Custody服务并将其顺利融入到他们的应用程序中，减少钱包开发所需的工作量。
</Accordion>
<Accordion title="权限控制和定制工作流">
ChainUp Custody托管服务提供基于角色的访问控制，并根据客户组织的规模和需求，有效地分隔用户权限，同时允许用户配置影响交易处理和批准方式的风控策略。
</Accordion>
<Accordion title="高效财务对账">
ChainUp Custody提供清晰可视的资产面板，专业全面的财务数据和相关明细表，支持客户获取实时的资产变化数据来帮助内部审计，实现资金全路径监控。
</Accordion>
<Accordion title="交易风控管理">
ChainUp Custody的策略引擎允许企业定制精细的事务和用户控制，以满足治理和风险要求，实现实时交易监控。用户可以根据需要，自定义审批工作流程，包括用户角色、交易类型、资产类型等风控维度。
</Accordion>
