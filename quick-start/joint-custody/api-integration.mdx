## I. 概述

Keysecure提供了强大且易于集成的 API工具，以自动化您的工作流程、快速创建并配置钱包并接收有关钱包各类信息。

## II. 我可以使用 Keysecure API 做什么？

**自动化钱包管理工作流程**

使用API快速创建和命名钱包，实现自动化的钱包管理。这使得批量创建和命名钱包的过程更加高效和便捷。

**自动化交易工作流程**

实现区块链交易的批量化处理，实现资产的自动化转移，并使用安全的API Co-Signer自动签署交易。

**扩展钱包和地址创建**

快速而经济高效地配置多个钱包。这对于游戏工作室、提供NFT服务或构建dApp的开发者来说，可以轻松管理大量的数字资产。

**快速查看交易信息**

快速获取钱包地址信息和交易明细，用户可以方便地获取全量的交易记录和明细，确保他们能够方便快捷地了解和监控其数字资产。

## III. 配置字段

您需要提供以下字段信息来完成API配置

**信任IP**

信任IP字段允许您定义可以访问API的特定IP地址范围。只有来自这些信任的IP地址发起的请求才会被API接受和处理。

**回调地址**

回调地址是API用于向您的系统发送充币、提币业务交易成功的地方。

**二次确认地址**

二次确认地址用于接收需要进一步确认或验证的交易或操作的通知。当商户发起提币请求后，平台向商户确认信息的地址。

**备注**

备注字段允许您添加额外信息或说明，通常用于标识特定的API配置。
