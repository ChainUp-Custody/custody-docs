## I. 功能简介

API管理功能允许钱包创建者查看和操作API相关信息。通过API管理页面，您可以输入API相关信息，并进行保存和配置，以确保安全的API通讯和交易。

## II. 操作流程

<Steps>

<Step title="API信息输入">
    只有钱包创建者有API管理页面的查看和操作权限。在API管理页面，您可以输入以下API相关信息：

    - **信任IP**：三方服务器的IP地址，添加后才可以跟Custody服务器进行正常通讯，否则即使拿到API信息也无法使用。
    - **回调地址**：有些业务无法实时反馈结果，后续处理完成主动向三方服务器通知处理结果的地址。
    - **二次确认地址**：提币业务二次确认地址，API发起提币请求后，系统会向该地址进行请求确认，进一步提升资金安全。

    ![API信息输入](/images/user-guide/api_management_input.png)
</Step>

<Step title="保存信息">
    输入完成后，请保存API信息，创建成功。目前仅支持创建一个全功能权限的API。

    ![保存信息](/images/user-guide/api_management_save.png)
</Step>

<Step title="API详情设置">
    点击API详情，设置开发者RSA公钥并获取系统RSA公钥。在填写开发者RSA公钥时，请在【----begin----】和【----end----】内填写相关数据。

    ![API详情设置](/images/user-guide/api_management_detail.png)
</Step>

</Steps>

## III. 注意事项

- 请确保输入的API信息准确无误，以确保API通讯和交易的安全性和准确性。
- API详情设置中的开发者RSA公钥填写需严格按照要求填写，以确保信息的正确传输和安全性。
