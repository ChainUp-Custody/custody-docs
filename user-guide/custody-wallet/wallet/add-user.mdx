## I. 功能简介

<Note>只有钱包管理员才可以添加/删除钱包成员。</Note>
在托管钱包中，您可以添加新的用户，并为他们分配不同的角色。目前系统有三类角色：

- **管理员** ：创建钱包的成员默认为钱包管理员，具有钱包所有权限；
- **审批成员** ：管理员将普通成员添加到审批策略后自动成为审批成员，具有发起收款、转账、审批等权限；
- **普通成员** ：管理员添加到钱包的成员默认为普通成员；具有发起收款、转账等基础权限；


<Tip>
**成为托管成员需满足：**
1. 拥有ChainUp Custody账号
2. 需要完成KYC认证
3. 绑定并开启谷歌验证器
</Tip>

## III. 常见问题

<Accordion title="如何修改用户的角色？">
管理员可以在钱包设置中修改用户的角色。请注意，只有钱包的创建者才可以修改用户角色。
</Accordion>

<Accordion title="如果我不是钱包的创建者，我可以添加用户吗？">
抱歉，只有钱包的创建者才可以添加用户。如果您不是钱包的创建者，您无法添加新的用户。
</Accordion>

<Accordion title="我可以在一个钱包中设置多个管理员吗？">
抱歉，每个钱包只能有一个管理员。管理员是创建钱包的用户，他们拥有钱包的所有权限。
</Accordion>

<Accordion title="如果一个用户被删除，他们可以再次被添加吗？">
是的，如果一个用户被删除，他们可以被再次添加。请注意，只有钱包的创建者才可以添加用户。
</Accordion>
