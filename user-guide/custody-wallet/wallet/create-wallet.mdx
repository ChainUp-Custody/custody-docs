## I. 功能简介

专为企业用户设计，支持多成员多角色协作管理钱包，并提供出金策略和审批功能。作为钱包的创建者，您将拥有以下权限：

- **团队成员管理**：在PC端添加钱包管理团队成员，分配团队成员职责。共管成员（审批和普通）可以在APP中发起或审批提币，并查看审批进度和资金流水记录。
- **出金策略管理**：制订重要币种的出金策略，确保提币操作遵循审批流程，保障资金安全。
- **API提币地址管理**：在PC端管理API提币地址，设定提币白名单地址。

## II. 操作流程

打开Keysecure官网，下载[Keysecure App](https://keysecure.io/)，或直接扫描二维码下载。
