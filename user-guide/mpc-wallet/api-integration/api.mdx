## I. 功能简介

ChainUp Custody为用户提供了强大且易于集成的API工具，可以帮助您自动化工作流程、快速创建并配置钱包并接收有关钱包各类信息。无论是自动化钱包管理、交易流程，还是快速查看交易信息，都能通过API实现。

## III. 常见问题

<Accordion title="我可以使用ChainUp Custody API做什么？">
ChainUp Custody API可以帮助您实现以下功能：
- 自动化钱包管理工作流程：使用API快速创建和命名钱包，实现自动化的钱包管理。
- 自动化交易工作流程：实现区块链交易的批量化处理，实现资产的自动化转移，并使用安全的API Co-Signer自动签署交易。
- 扩展钱包和地址创建：快速而经济高效地配置多个钱包。这对于游戏工作室、提供NFT服务或构建dApp的开发者来说，可以轻松管理大量的数字资产。
- 快速查看交易信息：快速获取钱包地址信息和交易明细，用户可以方便地获取全量的交易记录和明细，确保他们能够方便快捷地了解和监控其数字资产。
</Accordion>

<Accordion title="如果我有多个IP地址，应如何配置？">
每个API最多绑定10个IP地址，多个IP之间使用半角逗号分隔
</Accordion>

</Accordion>
