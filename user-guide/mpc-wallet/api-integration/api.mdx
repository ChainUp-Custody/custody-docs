## I. 功能简介

Keysecure为用户提供了强大且易于集成的API工具，可以帮助您自动化工作流程、快速创建并配置钱包并接收有关钱包各类信息。无论是自动化钱包管理、交易流程，还是快速查看交易信息，都能通过API实现。

## II. 操作流程

<Steps>

<Step title="配置API">
您可以在Keysecure的Web端方便地配置MPC系统的API信息，实现更灵活的钱包管理。

<div className="w-full flex items-center justify-center">
    <img src="/images/user-guide/apicn.png"/>
</div>



<Warning>
**重要提示**
一个MPC工作站仅能配置一个API。为确保API功能生效，API配置需要与Co-Signer配置同步进行。只有钱包创建者才可配置API信息，编辑修改信息需要通过GA验证。请参考[API文档](/api-references/intro/custody-apis)，详细了解如何有效地对接API。
</Warning>

</Step>


<Step title="API字段填写">
请填写以下API信息并提交新增API申请，等待工作站策略审批员审批通过后，API才可生效。


* **绑定IP地址：** 指定API接口监听的IP地址，只允许特定IP范围的请求。

* **交易状态通知地址：** 用于接收交易状态通知的URL地址。

* **Co-Signer接口地址：** 指定Co-Signer服务的API接口地址，确保正常的Co-Signer服务调用。

* **Co-Signer RSA公钥：** 用于加密和解密相关数据。确保正确的RSA公钥有助于维护通信的安全性。

* **客户系统RSA公钥：** 用于加密敏感信息并确保只有客户系统能够解密。提供正确的RSA公钥有助于保护通信过程中的数据安全。

* **备注：** 记录与API配置相关的附加信息，方便管理员和开发者理解配置的特殊要求或约定。

## III. 常见问题

<Accordion title="我可以使用Keysecure API做什么？">
Keysecure API可以帮助您实现以下功能：
- **自动化钱包管理工作流程：** 使用API快速创建和命名钱包，实现自动化的钱包管理。
- **自动化交易工作流程：** 实现区块链交易的批量化处理，实现资产的自动化转移，并使用安全的API Co-Signer自动签署交易。
- **扩展钱包和地址创建：** 快速而经济高效地配置多个钱包。这对于游戏工作室、提供NFT服务或构建dApp的开发者来说，可以轻松管理大量的数字资产。
- **快速查看交易信息：** 快速获取钱包地址信息和交易明细，用户可以方便地获取全量的交易记录和明细，确保他们能够方便快捷地了解和监控其数字资产。
</Accordion>

<Accordion title="如果我有多个IP地址，应如何配置？">
每个API最多绑定10个IP地址，多个IP之间使用半角逗号分隔
</Accordion>

<Accordion title="我如何使用API文档？">
您可以参考[API文档](/api-references/intro/custody-apis)来详细了解如何有效地对接API。API文档提供了详细的API接口信息和使用方法，是您使用API的重要参考资源。
</Accordion>
