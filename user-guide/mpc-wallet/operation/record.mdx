## I. 功能简介

在ChainUp Custody工作站中，您可以查看、筛选和导出您有权限的交易信息。您可以根据币种名称，NFT合集名称，合约地址，钱包，币种网络对该工作站的交易信息进行筛选搜索。

## III. 常见问题

<Accordion title="我如何查看其他工作站的交易信息？">
您可点击页面顶部的MPC按钮，或是点击左侧工作站切换按钮，切换其他工作站后，再进入该工作站的交易信息模块进行查看。
</Accordion>

<Accordion title="我为什么无法查看，搜索某笔交易信息？">
请确认以下几点：

1. 您是否拥有该笔交易的权限：您需要拥有相关工作站/钱包的共管权限；
2. 如果是转账交易，请确认这笔交易是否已创建订单号；
3. 如果是收款交易，请确认该笔订单的TXID，以及Custody系统是否支持该币种；
4. 如果您想查看Fil相关信息：抱歉，我们暂时不支持用户在MPC PC平台查看Fil相关信息。

如果仍然存在相关问题，请联系在线客服寻求帮助。
</Accordion>

<Accordion title="为什么部分交易信息看起来不完整？">
可能的原因有：

1. 系统无法全部解析合约调用类的交易信息字段；
2. 部分字段和交易状态关联：比如只有上链成功的交易才会显示矿工费。

您可联系在线客服寻求更多帮助。
</Accordion>
