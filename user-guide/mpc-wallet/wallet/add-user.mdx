## I. 功能简介

<Note>
只有工作站创建者才可以添加/删除工作站成员。
</Note>

在MPC工作站中，您可以添加新的用户，并为他们分配不同的角色。目前系统有三类角色：

- **工作站创建者** ：创建工作站的成员默认为创建者（Owner），具有最高权限；
- **管理员** ：创建者可以将通讯录的成员添加到工作站，并将其分配为管理员，管理员有权限发起工作站下所有钱包的收款、转账、并将拥有工作站同权私钥，对转账交易进行签名和加速；
<Tip> 
目前，管理员暂时无法对除了手动转账签名和加速以外的私钥操作行为进行签名。
</Tip>
- **普通成员** ：创建者添加到钱包的成员默认为普通成员；具有发起收款、转账等基础权限；

具体成员角色说明可以参考[MPC用户角色介绍](/help-center/manage-mpc/user-setting/user-role)

<Tip>
**成为MPC工作站成员需满足：**
1. 拥有ChainUp Custody账号
2. 需要完成KYC认证
3. 绑定并开启谷歌验证器
</Tip>

## III. 常见问题

<Accordion title="如何修改用户的角色？">
创建者可以在工作站成员中查看用户的权限详情，并修改用户的角色。请注意，若该成员属于某个钱包的审批流，则他将自动拥有该钱包的共管权限且无法取消。
</Accordion>

<Accordion title="如果我不是工作站创建者，我可以添加用户吗？">
抱歉，只有工作站创建者才可以添加用户。
</Accordion>

<Accordion title="工作站的角色限制？">
您的工作站总成员数量限制与您购买的服务套餐相一致，但对于具体的角色并无限制。
</Accordion>

<Accordion title="管理员如何管理他的私钥？">
目前系统不支持管理员对其私钥进行备份和恢复。如果因更换设备等原因导致私钥片段丢失，请联系工作站创建者重新发送（在“工作站成员”中选择相应工作站，查看您的具体权限以进行操作）
</Accordion>
