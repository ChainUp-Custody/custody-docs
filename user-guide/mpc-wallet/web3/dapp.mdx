## I. 功能简介
随着对 Web3 可访问性和使用的需求不断增长，越来越多的用户希望直接使用 dApp 来管理他们的 NFT、获取奖励等。ChainUp Custody 的 Web3 钱包链接功能允许您的用户直接将其钱包地址连接到 Web3 应用程序 (dApp)，无需使用其他钱包或任何其他应用程序。

<Note> 
请确保您已开通相应主链的Web3类型钱包，否则将无法授权DApp使用。

若您的钱包不支持Dapp的主链，请联系客服团队进行开通。
</Note>

## III. 常见问题

<Accordion title="资产类型钱包支持链接web3应用吗？">
不，资产类型钱包不支持链接web3应用。因此，在您需要使用web3 DApp之前，请先创建web3钱包。
</Accordion>

<Accordion title="web3交易支持MPC加速吗？">
是的，web3交易支持MPC加速。如果交易10分钟未上链，您会收到交易加速待办。web3交易加速与提币加速操作一致。
</Accordion>

<Accordion title="任何人都可以使用web3钱包链接DApp吗？">
不，只有工作站创建者才可以使用web3钱包链接DApp。
</Accordion>
